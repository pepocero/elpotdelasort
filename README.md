# El Pot de la Sort

Aplicación web educativa para crear grupos aleatorios y seleccionar personas al azar en el aula. Diseñada para pizarras digitales, uso táctil y pantallas grandes.

## Funcionalidades
- Configuración de aulas con nombres y listas de alumnado.
- Generador de grupos aleatorios con reparto equilibrado.
- Selector de una persona al azar con animación.
- Persistencia en `localStorage`.
- Exportación a PDF.
- Reinicio total de datos.

## Uso
1. Abrir `index.html` en el navegador.
2. Crear una aula en la pestaña **Configuració d’aules**.
3. Generar grupos o escoger personas en las pestañas correspondientes.

## Tecnología
- HTML5
- CSS3 (animaciones y transiciones)
- JavaScript puro
